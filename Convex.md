# 基本理论
* 仿射集
  1. 是凸集
  2. 加和为1
* 凸集
  1. 加和为1
  2. [0,1]
* 凸锥，
  1. 是凸集 
  2. 加和为1 
  3. 大于等于0，过原点

# 概念

  * 球
  * 椭球
  * 多面体
  * 单纯形
* 仿射函数 y = Ax + b, R(m维)到R(n维)
* 透视函数，R(n+1维)到R(n维)

# 凸集定理
* 仿射函数之后仍然为凸集，反函数也成立
* 透视函数之后仍然为凸集，反函数也成立
* 线性分式函数（仿射+投影）之后仍然是凸集

# 凸函数等价定义
* 一维，定义域凸集，t * x + (1-t) * y
* 高维，定义域凸集，g(t) = f(x + v * t)
* 一维，定义域凸集，f(y) >= f(x) + f'(x) * (y - x)
* 二阶导数，定义域凸集，f''(x) >= 0
